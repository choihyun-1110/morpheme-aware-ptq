## 4. 오류 패턴 분석

수정 전 수치들이 실제 파일보다 일관되게 높음. 가능한 원인:
- 이전 Claude 세션이 eval 스크립트 없이 수치를 추정/hallucination
- 구버전 eval 파일이 존재했다가 삭제된 경우 (확인 불가)
- lm-eval 버전 차이로 다른 결과가 나온 이전 실행 존재 가능

현재 results/ 디렉토리에 있는 JSON 파일이 실제 측정값의 ground truth.
