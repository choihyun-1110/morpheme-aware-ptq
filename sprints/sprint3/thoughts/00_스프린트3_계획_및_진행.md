## S3-1: Qwen2-7B 중국어 calibration 실험 (진행 중)

### 완료된 작업

#### Chinese Calibration Set 구축 (`src/build_calibration_zh.py`)
- wikimedia/wikipedia 중국어 스트리밍 데이터 사용 (streaming=True)
- jieba posseg 형태소 분석 (중국어 단어 + 품사 태깅)
- 중국어 한자 비율 필터 (zh_char_ratio ≥ 0.7)
- Greedy 다양성 선택 (C_v3와 동일 알고리즘)
- 결과: **128문장, 1652 고유형태소, avg_SFS=2.29**
- 저장: `results/calibration_set_C_zh_Qwen_Qwen2-7B-Instruct.json`
- 소요 시간: 637초 (~10분)

#### Qwen2-7B 다운로드 완료
- 경로: `.cache/huggingface/hub/models--Qwen--Qwen2-7B-Instruct/snapshots/f2826a00ceef68f0f2b946d945ecc0477ce4450c/`
- 크기: ~12GB (safetensors x4)

#### 양자화 진행 (GPU 0, 순차 체인)
- [✅] 조건 A (영어 baseline): `quantized_models/qwen2_7b_cond_A`
- [🔄] 조건 C_v3 (한국어 형태소 다양성): 진행 중
- [⏳] 조건 C_zh (중국어 형태소 다양성): 대기 중

### 기술 메모
- `TRANSFORMERS_OFFLINE=1` 방식 실패 (캐시 탐색 실패)
- 해결책: `--model /full/path/to/snapshot` 직접 지정
- Qwen2-7B는 28 레이어 (SOLAR 48레이어보다 적어 빠름, ~20분/조건)
