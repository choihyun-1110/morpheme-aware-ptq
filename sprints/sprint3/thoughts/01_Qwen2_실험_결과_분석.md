## 해석 및 새로운 통찰

### 핵심 관찰

1. **C_v3(한국어) > A(영어)**: 한국어 calibration이 영어보다 유리 — KoBEST가 한국어 벤치마크이므로 당연한 결과일 수 있음

2. **C_zh(중국어) < A(영어)**: Qwen2의 사전학습 주 언어인 중국어 calibration이 가장 낮음 — 가설과 정반대

3. **hellaswag에서 C_zh 급락**: A=0.5260, C_v3=0.5260, **C_zh=0.4420** — 중국어 calibration이 GPTQ 추론 능력을 왜곡

### 가설 재고

Sprint 2에서 발견한 원리:
> "사전학습 주 언어 = calibration 언어 → GPTQ 품질 보존"

이 원리를 적용하면 Qwen2에서 C_zh가 최우수여야 하지만, 실제로는 C_v3가 최우수.

**두 가지 대안적 해석:**

**해석 A (평가 언어 정렬):**
> "calibration 언어 = 평가 언어 → 해당 언어 능력 보존"
- KoBEST = 한국어 평가 → C_v3(한국어) 최우수
- C_zh가 Qwen2의 중국어 능력은 보존하지만 한국어 능력은 더 왜곡

**해석 B (형태소 다양성의 범언어적 효과):**
> C_v3의 형태소 다양성 알고리즘이 언어와 관계없이 더 다양한 activation을 유도
- 즉, SOLAR에서 C_v3 > B인 원인(비한국어 subword 제거)이 Qwen2에서도 적용
- C_zh는 중국어 특화라 Qwen2의 한국어 처리 레이어를 충분히 커버하지 못함

### SOLAR vs Qwen2 비교

| 모델 | 사전학습 | 최우수 calibration | 비고 |
|------|----------|-------------------|------|
| SOLAR-10.7B | 한국어 | C_v3 (한국어) | 가설 지지 |
| Llama3-Ko-8B | 영어 | A (영어) | 가설 지지 |
| EEVE-10B | 한국어(SFT) | B (한국어 랜덤) | 부분 지지 |
| Qwen2-7B | 중국어 | C_v3 (한국어) | **가설 반증** |

## 결론

Qwen2 실험 결과는 **"사전학습 언어 = calibration 언어"** 단순 가설을 지지하지 않는다.

오히려:
1. **calibration 언어 = 평가 언어** 원리가 더 설명력이 있을 수 있음
2. **형태소 다양성 자체**가 언어를 초월하는 핵심 요소일 가능성

→ 논문 주장 수정: "calibration 언어 정렬"보다 "형태소 다양성 최적화"가 핵심
→ 중간 보고서에서 이 갈림길을 연구 질문으로 명시 필요

## [추가] C-Eval 결과 (2026-03-26)


C-Eval (중국어 벤치마크)로 Qwen2 GPTQ 조건별 성능 측정 완료.

### Qwen2-7B C-Eval (ceval-valid) 결과

| 조건 | C-Eval acc | KoBEST avg | 비고 |
|------|-----------|-----------|------|
| FP16 (베이스라인) | **0.8165** | - | 미측정 |
| A (영어 Wikitext) | 0.7630 | 0.6238 | C-Eval 보존율 93.4% |
| C_v3 (한국어 형태소) | **0.7734** | **0.6375** | C-Eval 보존율 **94.7%** |
| C_zh (중국어 형태소) | 0.7630 | 0.6063 | C-Eval 보존율 93.4% |

**순위 (C-Eval): C_v3 > A = C_zh**

### 핵심 발견

1. **C_zh는 중국어 벤치마크에서도 A와 동점** — Qwen2의 사전학습 언어로 만든 calibration이 해당 언어 벤치마크에서조차 유리하지 않음
2. **C_v3(한국어)가 C-Eval에서도 최우수** — 형태소 다양성 기반 선택의 효과가 언어 무관
3. KoBEST와 C-Eval 모두 동일한 패턴: **C_v3 > A ≥ C_zh**

### 가설 최종 판단

| 가설 | 결과 |
|------|------|
| "사전학습 언어 = calibration 언어 → 최우수" | **기각** (C_zh 최하 또는 동점) |
| "calibration 언어 = 평가 언어 → 최우수" | **부분 기각** (C_v3가 C-Eval에서도 우세) |
| "형태소 다양성 자체가 핵심 요소" | **지지** (C_v3의 높은 SFS가 언어 무관하게 효과적) |

### 결론

Qwen2 실험은 **SOLAR 결과의 일반화 가능성**을 보여준다:
- C_v3가 SOLAR(한국어 사전학습)뿐 아니라 Qwen2(중국어 사전학습)에서도 최우수
- 단, Llama3-Ko(A 최선), EEVE(B 최선)와 비교하면 "모델 아키텍처/학습 방식"에 따라 다름
- C_v3의 핵심: **subword 효율성 + 형태소 다양성** → 대부분 모델에 긍정적 영향

→ 논문 주장: "morpheme-diverse Korean calibration set이 언어/모델 종류를 초월해 GPTQ 왜곡을 최소화"
